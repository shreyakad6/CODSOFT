# Task 3: Restaurant Menu UI/UX Design

This project focuses on designing a clean and user-friendly restaurant menu interface that allows users to easily browse food items and place orders.

## Screens Included
- Menu Home Screen
- Food List Screen
- Item Details Screen

## Design Highlights
- Clear food categorization
- Visually appealing food cards
- Easy-to-use quantity selector
- Prominent call-to-action buttons

## UX Focus
- Simple navigation
- Minimal and clean layout
- Consistent colors and typography
- Mobile-friendly design

- Figma
